## 📋 使用情境

本工具基於 SHARC 談判個案研究開發，用於模擬四個漁業協會在不同捕撈量協議下的淨現值（Net Present Value）結果，並透過最佳化演算法自動尋找使所有協會總 NPV 最大化的捕撈量組合。

> 💡 **適用對象**：談判課程教學、利益相關者決策分析、協議方案評估

---

## 🎯 個案背景

沿海大鯊魚總捕撈量需從目前的 **5,000 公噸**減少至 **2,500 公噸**，以維持鯊魚族群的長期安全存活量。四個利益相關協會需協商各自明年的捕撈配額。

### 未來捕撈量公式

```
未來捕撈量 = min(2500, 5000 − 四協會捕撈量總和)
```

---

## 👥 四個協會參數

| 協會 | 代號 | 目前捕撈量 | 最低配額 | 未來依賴係數 | NPV 公式 |
|------|------|-----------|---------|------------|---------|
| 大型商業漁民協會 | **LC** | 2,000 公噸 | 400 公噸 | 0.10 | `$10,000 × (LC量 + 0.10 × 未來量)` |
| 小型商業漁民協會 | **SC** | 1,500 公噸 | 300 公噸 | 0.25 | `$10,000 × (SC量 + 0.25 × 未來量)` |
| 休閒比賽漁民協會 | **RC** | 1,000 公噸 | 200 公噸 | 0.70 | `$10,000 × (RC量 + 0.70 × 未來量)` |
| 休閒旅遊漁民協會 | **RT** |   500 公噸 | 100 公噸 | 0.85 | `$10,000 × (RT量 + 0.85 × 未來量)` |

### 限制條件

- ✅ 明年配額不可超過目前捕撈量（歷史高點上限）
- ✅ 最低配額為目前水準的 **20%**（維持基本營運底線）
- ✅ 各協會間不得進行漁獲或利益交換

## ✨ 功能特色

### 📊 即時 NPV 計算
- 拖拉滑桿或直接輸入數字，數值即時更新
- 未來捕撈量依 `min(2500, 5000 - 總量)` 自動計算
- 各協會 NPV 同步顯示，可見交互影響
- 狀態列即時顯示協議總量、未來捕撈量、各方 NPV 總和

### 🔒 雙模式模擬切換
| 模式 | 說明 |
|------|------|
| 🔓 **無總量限制**（預設） | 各協會在各自允許範圍內自由設定，觀察 NPV 變化 |
| 🔒 **2,500 公噸總量上限** | 啟用後，四協會總量 > 2,500 公噸時即觸發紅色警告橫幅，最佳化計算也受此限制 |

> 切換按鈕位於狀態列正下方，一鍵切換，即時生效。

### ⚠️ 總量超限即時警告
啟用上限模式後，當協議總捕撈量超過 2,500 公噸時：
- 出現動態紅色警告橫幅（含動畫）
- 顯示目前總量、超出量、以及視覺化進度條
- 狀態列的「協議總捕撈量」文字同步變紅提示

### 📈 多維度圖表

| 圖表 | 說明 |
|------|------|
| NPV 橫條圖 | 各協會淨現值即時比較 |
| 捕撈量分配圓餅圖 | 現況 vs 明年規劃分配對比 |
| 係數敏感度折線圖 | 未來捕撈量對各協會 NPV 的影響趨勢 |
| LC × SC 熱圖 | 兩變數組合下總 NPV 的視覺化分佈 |

### ⚡ 最佳化模擬（雙模式）
- 窮舉法搜索所有合法捕撈量組合（步長 50 公噸）
- **無限制模式**：找出各方 NPV 總和最大的全域最佳解
- **2,500 上限模式**：在總量 ≤ 2,500 公噸的限制下找最佳解
- 結果標示當次計算所使用的模式（🔒 / 🔓）
- 一鍵「套用最佳解」回輸入面板

### 📱 自適應設計 (RWD)
- **桌面（> 900px）**：左右雙欄佈局，完整功能
- **平板（≤ 900px）**：縮排版式，2×2 摘要卡片
- **手機（≤ 600px）**：單欄堆疊，大型觸控滑桿，圓餅圖垂直排列

## 📐 核心計算公式

```javascript
// 未來捕撈量
futureCatch = Math.min(2500, 5000 - (LC + SC + RC + RT))

// 各協會 NPV
LC_NPV = 10000 * (LC + 0.10 * futureCatch)
SC_NPV = 10000 * (SC + 0.25 * futureCatch)
RC_NPV = 10000 * (RC + 0.70 * futureCatch)
RT_NPV = 10000 * (RT + 0.85 * futureCatch)

// 2,500 上限模式下的最佳化篩選條件
if (LC + SC + RC + RT > 2500) skip;  // 不納入搜索
```
